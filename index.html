Tentang Saya

Nama saya Chico Endar Adikara, seorang siswa kelas X di SMKN 9 Semarang, jurusan Pengembangan Perangkat Lunak & Game. Saya sangat antusias mempelajari teknologi dan bagaimana cara menciptakan perangkat lunak serta game yang inovatif. Di sekolah, saya fokus memperdalam HTML, CSS, dan JavaScript sebagai pondasi dasar pengembangan web.

Selain belajar di kelas, saya juga aktif di ekstrakurikuler Robotika. Melalui kegiatan ini, saya belajar tentang teknologi otomatisasi dan pemrograman hardware, yang menurut saya sangat menarik dan relevan dengan pengembangan perangkat lunak. Robotika juga membantu saya memahami logika pemrograman dengan cara yang lebih nyata.

Saya selalu berusaha mengembangkan kemampuan saya di bidang pengembangan perangkat lunak dan game. Dengan minat yang kuat di teknologi, saya ingin menciptakan proyek-proyek yang bermanfaat serta terus belajar lebih dalam tentang dunia pemrograman. Kedepannya, saya berharap dapat berkontribusi lebih dalam industri teknologi dan game development.
